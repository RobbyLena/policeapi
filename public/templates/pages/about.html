About this API and website

Opendata example, attempt to make difference, help those, who lost their vehicles. Help those, who can accidentally buy stolen car. Your chance to verify, if vehicle is registred as stolen in Czech Republic.

There are more accessible databases. I know about Romanian, Slovenian, which can be utilized same way as Czech. It would be good to have one API that uses as many as possible national databases. I will do my best to incorporate those databases to my API. Unfortunately i do not have any sponsor (or a few milions of taxpayers) behind my backs, so forgive me slow development and responses. I do it in my spare time.

Technologies

I am programmer, web developer. I love all the new technologies and services. This was a chance to try them. Learn something new. It's all opensource or free, so you can use it too.

Node.js

JavaScript backend. Fast, nonblocking, ideal for my API.

Used modules (from nmp):
  • express - fast webapp development
  • request - HTTP requests to police website
  • cheerio - jQuery like HTML parsing
  • easyxml - JSON to XML transformation
  • mongodb - data storage
  • jasmine - test framework

OpenShift

Red Hat platform as service. Environment which powers this web and API. FreeShift variant gives me enough space and performance for now.

Apiary.io

API documentation, testing. Describe API is not easy. Apiary make it as easy, as possible.

MongoDB

NoSQL database, storage for API data. Document oriented database. Json is base storage format, so is very easy just to put there API call results.

Twitter Bootstrap

Front-end web responsive framework. Easy to build nice website even for programmer without any graphical sense. Responsive is great benefit out of box (Don't know? just try to resize this page).

Travis-CI

Hosted continuous integration service for open source and private projects. Test runner for jasmine tests in this project. Configured to test every new revision found in git repository. Current build status:
