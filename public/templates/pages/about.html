About this API and website

Technology toy, opendata example, my hobby. Your chance to verify, if vehicle is registred as stolen in Czech Republic.

Technologies

I am programmer, web developer. I love all new technologies, services and utilities. So i made this project based on this technologies:

Node.js

JavaScript backend. Fast, nonblocking, ideal for my API.

Used modules (nmp):
  • express - fast webapp development
  • request - HTTP requests to police website
  • cheerio - jQuery like HTML parsing
  • easyxml - JSON to XML transformation
  • mongodb - data storage

OpenShift

Red Hat platform as service. Environment which powers this web and API.

Apiary.io

API documentation, testing.

MongoDB

NoSQL database, storage for API data. Document oriented database. Json is base storage format, so is very easy just to put there API call results.

Twitter Bootstrap

Front-end responsive framework. Easy to build nice website even for programmer without any graphical sense. Responsive is great benefit out of box (Dont know? just try to resize this page).

And many more tools and technologies. Versioned in Git, Intellij IDEA as IDE, jQuery for frontend JavaScript...
