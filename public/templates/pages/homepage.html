API k policejní databázi odcizených vozidel

Policie ČR neposkytuje žádné API pro dotazování na odcizená vozidla. Do konce roku 2011 poskytovali pěkné XML API. Pak jej ale bez náhrady zrušili a myslí si, že všem musí stačit formulář na jejich webu.

Pokud si myslíte něco jiného, stejně jako já, pak jste tu dobře. Vytvořil jsem jednoduché API postavené nad webovým formulářem, které vám zpříjemní vyhledávání a ověřování odcizených vozidel. Nemusíte tak sami parsovat nepříjemné HTML a starat se o aktualizaci scriptů v okamžiku, kdy policie změní strukturu webu.
