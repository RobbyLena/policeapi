Unofficial API to official Czech police database of stolen vehicles.

Czech police doesn't understand the concept of open data. They does not offer any API to search and verify vehicle identificators. Once there was one API, but at end the of 2011, diappeared without replacement. So i decided to create API based od official web search located on police website.

When I asked, why there is no API, I got response (translated):

"Your automatic processing is useless, if you can find current data on our web page online."

So I do exactly what they said. I'm searching data on their website and provide them in JSON or XML.

Police has no idea, that it can be possible to verify vehicles in real time in our systems. Everywhere, where you process any identificators of vehicles, you can check, if it's not stolen. They don't know, what API is and why is useful.
All the details of my communication with Czech police is here (in Czech language).
