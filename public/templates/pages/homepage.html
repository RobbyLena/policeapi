API k policejní databázi odcizených vozidel

Policie ČR neposkytuje žádné API pro dotazování na odcizená vozidla. Do konce roku 2011 poskytovali pěkné XML API. Pak jej ale bez náhrady zrušili a myslí si, že všem musí stačit formulář na jejich webu.

Když jsem se ptal proč, dostalo se mi odpovědi:

"Vaše automatické zpracování dat v systémech je zbytečné, pokud tu nejaktuálnější verzi databáze máte na našich stránkách on line."

Policie zřejmě vůbec nechápe, že bylo možné ihned ověřovat odcizenost vozidla všude tam, kde se pracuje s SPZ / RZ nebo VIN. Netuší, k čemu jsou API a jak mají vypadat dostupná data. Celý příběh si můžete přečíst na odcizena-vozidla.cz.

Pokud si myslíte, stejně jako já, že by informace o odcizeném vozidle měla být co nejsnáz dostupná, pak jste tu dobře. Vytvořil jsem jednoduché API postavené nad oficiální webovým formulářem, které vám zpříjemní vyhledávání a ověřování odcizených vozidel. Nemusíte tak sami parsovat ošklivé HTML a starat se o aktualizaci scriptů v okamžiku, kdy policie změní strukturu webu.

Každým vyhledáváním pomáháte. Nejen, že zjistíte, zda je vozidlo hledané. Každým dotazem pomáháte zpřístupnit část policejní databáze vyhledavačům. Pro roboty je web policie neprůchozí - to je špatně. Když se ale kdekoliv na internetu objeví odkaz na detail vozidla v policejní databázi, rázem o stránce budou vědět všechny vyhledavače (seznam, google, bing,...) a mohou ji zahrnout do výsledků vyhledávání.
